# Decision Tradeoff Skill

Use this skill when the user needs a recommendation between multiple options.

## Workflow

1. Identify the decision to be made.
2. List the relevant options.
3. Evaluate each option by benefits, costs, risks, reversibility, and operational complexity.
4. Recommend one option and explain why.

## Rules

- Prefer explicit trade-offs over vague pros and cons.
- Weight user constraints more heavily than generic best practices.
- Say when two options are effectively equivalent.

End of Skill.

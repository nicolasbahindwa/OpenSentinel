# Uncertainty Skill

Use this skill when the input is incomplete, ambiguous, or based on weak evidence.

## Workflow

Classify important statements as:

- Fact: directly supported by evidence or context
- Inference: conclusion drawn from facts
- Assumption: chosen working premise
- Unknown: cannot be determined reliably

## Rules

- Keep uncertainty explicit but concise.
- Do not hide missing information behind confident wording.
- Use the minimum number of assumptions needed to keep moving.
- If an assumption is risky, surface it before acting.

End of Skill.

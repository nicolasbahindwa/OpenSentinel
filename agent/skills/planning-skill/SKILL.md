# Planning Skill

Use this skill for multi-step work where ordering and validation matter.

## Workflow

1. Define the goal in one sentence.
2. Split the work into the smallest meaningful steps.
3. Mark blockers, dependencies, and validation points.
4. Do the next blocking step first.
5. Revise the plan if new facts change scope.

## Rules

- Keep the plan outcome-oriented.
- Do not spend more time planning than the task justifies.
- Pair major steps with validation when possible.

End of Skill.
